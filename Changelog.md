# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Изменено
- **Секция "ТЕКУЩИЙ ФОКУС"**:
  - Перенесена в конец портфолио (после секции "ЦЕЛИ И ПЛАНЫ")
  - Убрано центрирование текста, применено стандартное выравнивание по левому краю
  - Обновлено содержимое секции с новой структурой:
    - Добавлены списки с маркерами для лучшей читаемости
    - Структурирована информация о текущих фокусах и открытости к сотрудничеству
  - Добавлен новый раздел "Оборудование" с информацией о рабочей станции
  - Добавлена точка после "Macbook" в описании оборудования
  - Ссылка на `hardware.md` подготовлена для последующей вставки рабочего URL

### Стили
- Удалены старые стили `.focus-banner` с центрированием
- Добавлены новые стили для секции `#focus`:
  - `.focus-content` - контейнер для основного содержимого
  - `.focus-list` - стили для списков с маркерами
  - `.equipment-section` - стили для раздела оборудования
- Добавлена поддержка темной темы для всех новых элементов
- Добавлены адаптивные стили для мобильных устройств

### Файлы
- `index.html`: Обновлена структура секции "ТЕКУЩИЙ ФОКУС"
- `css/main.css`: Добавлены стили для новой структуры секции

## Формат записей

- **Добавлено** - для новых функций
- **Изменено** - для изменений в существующей функциональности
- **Устарело** - для функций, которые скоро будут удалены
- **Удалено** - для удаленных функций
- **Исправлено** - для исправления ошибок
- **Безопасность** - в случае уязвимостей
