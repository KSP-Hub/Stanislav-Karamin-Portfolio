# Оборудование и рабочие станции

## Основная рабочая станция (PC)

**Модель:** Dell Precision 7920 Tower  
**Операционная система:** Windows 10 Pro для рабочих станций (также доступны Linux-сборки)

### Технические характеристики

| Компонент | Спецификация |
|-----------|--------------|
| **Процессор** | 2× Intel Xeon Silver 4210R (10 ядер/20 потоков каждый) |
| **Тактовая частота** | 2.40 GHz (базовая) |
| **Оперативная память** | 128 GB DDR4 ECC |
| **Накопители** | • 4 TB HDD (3.64 TB доступно)<br>• 1 TB NVMe SSD (PC711 SK hynix) |
| **Графический ускоритель** | NVIDIA Quadro RTX 4000 (8 GB GDDR6) |
| **Архитектура** | x64 |

### Ключевые особенности
- Серверная архитектура с двумя процессорами Xeon
- ECC-память для повышенной стабильности вычислений
- Профессиональная видеокарта Quadro с поддержкой RTX
- Высокоскоростной NVMe-накопитель
- Оптимизирована для рабочих станций (CAD, 3D рендеринг, ML)

## Дополнительные устройства

- **Linux ноутбук** — для кроссплатформенной разработки и тестирования
- **MacBook** — для мобильной работы и тестирования в экосистеме Apple
